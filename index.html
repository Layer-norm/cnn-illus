核心思想

局部感受野

每个神经元只关注输入的一个局部区域，而非全部像素。这大幅减少了参数数量，并保留了空间结构信息。

参数共享

权值共享机制

同一个卷积核在整个输入上滑动，所有位置共享相同的权重参数，使网络具有平移不变性。

层级特征

层次化特征提取

浅层学习边缘、纹理等低级特征，深层学习形状、物体等高级语义特征，逐层抽象。

卷积层 Convolution Layer

卷积层是 CNN 的核心组件，通过卷积核在输入上滑动来提取局部特征

📐 CNN 各层输出尺寸计算公式

O = ⌊(I − K + 2P) / S⌋ + 1
其中 O = 输出尺寸，I = 输入尺寸，K = 卷积核大小，P = 填充大小，S = 步幅
符号含义典型值
I输入特征图的空间尺寸 (高或宽)32, 64, 224 ...
K卷积核 / 池化窗口大小3, 5, 7
P零填充 (padding) 大小0, 1, 2
S滑动步幅 (stride)1, 2

关键超参数

卷积层超参数详解

超参数说明影响
卷积核大小 (Kernel Size)常见 3×3、5×5、7×7、1×1越大感受野越大，参数越多；3×3 最常用
步幅 (Stride)卷积核每次滑动的距离Stride > 1 可下采样，减小特征图尺寸
填充 (Padding)输入边缘填充零的层数Same 填充保持输出尺寸不变
卷积核数量 (Filters)决定输出通道数（特征图数量）越多提取特征越丰富，但计算量增大
膨胀率 (Dilation)卷积核元素之间的间距增大感受野而不增加参数

池化层 Pooling Layer

池化层用于降低特征图的空间维度，减少计算量并增强特征的平移不变性

最大池化

Max Pooling

y(i, j) = max(m,n)∈Rij x(m, n)
取池化窗口 R 内的最大值作为输出，保留最显著的特征

Max Pooling 是最常用的池化方式。它选取局部区域中的最大值，能有效保留纹理和边缘等强特征响应。

平均池化

Average Pooling

y(i, j) = (1 / |Rij|) · Σ(m,n)∈Rij x(m, n)
取池化窗口 R 内所有值的平均值，保留区域整体信息

Average Pooling 计算局部区域的平均值，保留更多背景信息。常用于网络最后一层做全局平均池化 (GAP)。

池化层特性

池化层的关键特点

📉

降维减参

将特征图尺寸缩小，减少后续层的计算量和参数量

🔄

平移不变

微小位移不影响池化结果，增强特征的空间鲁棒性

🚫

无学习参数

池化操作不包含可训练参数，仅做固定的聚合运算

激活函数 Activation

激活函数为网络引入非线性，使 CNN 能够学习复杂的特征映射

f(x) = max(0, x)
输出 = max(0, 输入)，负值归零，正值不变
优点：计算简单高效，有效缓解梯度消失问题，是 CNN 中最常用的激活函数。
缺点：负值区域梯度为零（神经元死亡），输出非零中心化。
f(x) = 1 / (1 + e−x)
将任意实数映射到 (0, 1) 区间
优点：输出在 (0,1)，适合概率输出和二分类。
缺点：易梯度消失，非零中心化，含有指数运算（计算较慢）。
f(x) = (ex − e−x) / (ex + e−x)
将任意实数映射到 (−1, 1) 区间，零中心化
优点：零中心化输出，梯度比 Sigmoid 更强。
缺点：仍有梯度消失问题，含有指数运算。
f(x) = x  if x > 0,  αx  if x ≤ 0
α 通常取 0.01，负值区域有微小梯度
优点：解决了 ReLU 神经元死亡问题，负值区域仍有梯度。
缺点：α 需要手动设定，效果不一定优于 ReLU。

全连接层 & 输出层

全连接层将卷积层提取的特征映射到最终的分类或回归结果

全连接层

Fully Connected Layer

y = σ(Wx + b)
W 为权重矩阵，x 为展平后的特征向量，b 为偏置，σ 为激活函数
1

特征展平 (Flatten)

将卷积/池化输出的多维特征图 (H×W×C) 展平为一维向量 (H×W×C,)

2

线性变换

通过权重矩阵 W 和偏置 b 对特征向量做仿射变换 y = Wx + b

3

非线性激活

应用 ReLU 等激活函数引入非线性，增强网络表达能力

输出层

Softmax 输出

P(y=k|x) = ezk / Σj=1C ezj
z 为最后一层输出的 logits，C 为类别数，输出为各类别的概率分布
交叉熵损失：L = −Σk=1C yk · log(Pk)，其中 y 为 one-hot 编码的真实标签
全局平均池化 (GAP)：现代网络常在最后一层卷积后使用 GAP 替代 Flatten+FC，大幅减少参数量。GAP 将每个通道取平均值，输出维度 = 通道数。

代码示例

使用 PyTorch 构建一个简单的 CNN 分类网络

import torch
import torch.nn as nn

class SimpleCNN(nn.Module):
  def __init__(self, num_classes=10):
    super(SimpleCNN, self).__init__()

    # 卷积块 1: Conv → ReLU → MaxPool
    self.conv_block1 = nn.Sequential(
      nn.Conv2d(3, 32, kernel_size=3, padding=1),
      nn.ReLU(inplace=True),
      nn.MaxPool2d(kernel_size=2, stride=2)
    )

    # 卷积块 2: Conv → ReLU → MaxPool
    self.conv_block2 = nn.Sequential(
      nn.Conv2d(32, 64, kernel_size=3, padding=1),
      nn.ReLU(inplace=True),
      nn.MaxPool2d(kernel_size=2, stride=2)
    )

    # 全连接分类头
    self.classifier = nn.Sequential(
      nn.Flatten(),
      nn.Linear(64 * 8 * 8, 128),
      nn.ReLU(inplace=True),
      nn.Linear(128, num_classes)
    )

  def forward(self, x):
    x = self.conv_block1(x)    # [B,3,32,32] → [B,32,16,16]
    x = self.conv_block2(x)    # [B,32,16,16] → [B,64,8,8]
    x = self.classifier(x)     # [B,64,8,8] → [B,num_classes]
    return x
